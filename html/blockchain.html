  ¿Qué es el block-chain?
Simplemente, es un grupo de individuos controla una cadena de bloques como una base de datos distribuida para almacenar y compartir información. Estos usuarios independientes mantienen la integridad de los datos sin ningún control central. Nadie es dueño de Blockchain. Este hecho hace que Blockchain sea tan popular. Pero, aunque se la conoce como base de datos, no ha tenido todas las características que normalmente tiene una base de datos. un grupo de individuos controla una cadena de bloques como una base de datos distribuida para almacenar y compartir información.   Tipos de block-chains
Se utilizan ampliamente tres tipos de Blockchain.
  • Cadena de bloques pública: cualquiera puede unirse a la red y crear, escribir y participar en la red.
  • Cadena de bloques privada: este tipo de cadena de bloques está restringida a un grupo particular de personas.
  • Consortium Blockchain: al compartir recursos y costos con otros, algunas organizaciones se unen con el fin de lograr un objetivo común.
 
  Bloque y cadena
Los bloques son el contenedor de transacciones válidas. Además, almacena algunos tipos de valor hash que se utilizan para vincular un bloque a otros bloques. Hay principalmente dos valores hash en un bloque. Uno de ellos es el hash del bloque anterior y otro es el propio. 
  Hash
Hash es una cadena de longitud fija que es única para un bloque en particular. Es generado por una función Hash. La función hash tomará cualquier cosa y la convertirá en una cadena de longitud fija. Es un proceso unidireccional, lo que significa que no puede revertir la cadena hash a la entrada. El hash también se conoce como la huella digital del bloque.
